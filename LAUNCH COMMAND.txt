AI Orchestrator – Launch Commands (Updated)

Option A: One‑off run with inline API key (recommended quick start)

Linux / macOS (bash / zsh)
OPENAI_API_KEY="sk-yourkey" python3 orchestrator.py "Make me a GUI calculator"

Windows PowerShell
$env:OPENAI_API_KEY = "sk-yourkey"; python orchestrator.py "Make me a GUI calculator"

Windows CMD
set OPENAI_API_KEY=sk-yourkey && python orchestrator.py "Make me a GUI calculator"


Option B: Configure once, then run (interactive setup)

1) Create venv, install deps, and configure your API key:
	python3 setup.py

	During setup you can choose:
	- Environment variable
	- System keyring (secure, persistent)
	- Config file (~/.config/ai_orchestrator/openai_api_key)
	- Local file (development only)

2) Run the orchestrator using the venv Python shown at the end of setup, for example:
	Linux / macOS:
	./\.venv/bin/python orchestrator.py "Make me a GUI calculator"

	Or activate first:
	source ./.venv/bin/activate
	python orchestrator.py "Make me a GUI calculator"

	Windows (PowerShell):
	.\.venv\Scripts\python.exe orchestrator.py "Make me a GUI calculator"
   
	Or activate first:
	.\.venv\Scripts\Activate.ps1
	python orchestrator.py "Make me a GUI calculator"


Notes
- You can omit the prompt argument and type it interactively when asked.
- Useful flags:
  --policy/-p [A|S|D|K]   Save policy (Auto/Save/Delete/Keep)
  --model MODEL           LLM model (default from config)
  --temperature FLOAT     LLM temperature (default from config)
  --max-retries INT       Max retries for generation
  --no-run                Do not auto-run the generated project
  --verbose/-v            Enable verbose logging

Examples
OPENAI_API_KEY="sk-yourkey" python3 orchestrator.py -p S --no-run "Make a CLI countdown timer"
python3 orchestrator.py --verbose "Make a Flask hello world app"
